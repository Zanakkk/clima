## 5. PANDUAN PENGGUNAAN (STEP-BY-STEP)

### 5.1 Menambah Pasien Baru

1. Login ke akun CLIMA Anda
2. Dari dashboard, klik menu **Data Pasien**
3. Pilih tombol **+ Tambah Pasien Baru**
4. Isi formulir dengan data pasien:
    * Data pribadi (nama, tanggal lahir, dll)
    * Informasi kontak
    * Riwayat medis singkat
    * Informasi asuransi (jika ada)
5. Upload foto pasien (opsional)
6. Klik **Simpan** untuk menyimpan data
7. Sistem akan otomatis membuat nomor rekam medis unik

### 5.2 Mencatat Janji Temu

1. Dari dashboard, buka menu **Janji Temu**
2. Klik **+ Buat Janji Baru**
3. Pilih pasien dari database (ketik nama untuk pencarian cepat)
4. Pilih dokter yang akan menangani
5. Tentukan tanggal dan jam kunjungan
6. Pilih jenis layanan/tindakan
7. Tambahkan catatan khusus jika diperlukan
8. Klik **Simpan Janji**
9. Sistem akan otomatis mengirimkan pengingat ke pasien

### 5.3 Input Tindakan Medis

1. Dari dashboard, masuk ke menu **Riwayat Tindakan**
2. Cari dan pilih pasien yang ditangani
3. Klik **+ Tambah Tindakan**
4. Isi detail tindakan:
    * Jenis tindakan (pilih dari katalog)
    * Dokter yang menangani
    * Deskripsi tindakan
    * Hasil dan catatan medis
5. Input biaya tindakan (atau sistem akan menghitung otomatis)
6. Upload foto kondisi jika diperlukan
7. Klik **Simpan Tindakan**
8. Data akan tersimpan dalam rekam medis pasien

### 5.4 Mengakses Analitik Data

1. Dari dashboard, klik menu **Analitik**
2. Pilih jenis laporan yang diinginkan:
    * Statistik pasien
    * Performa dokter
    * Laporan keuangan
    * Tren kunjungan
3. Tentukan rentang waktu laporan (hari/minggu/bulan/tahun)
4. Sistem akan menampilkan visualisasi data dalam bentuk grafik
5. Untuk ekspor data, klik tombol **Download** dan pilih format (PDF/Excel)

### 5.5 Manajemen Payroll

1. Dari dashboard, buka menu **Payroll**
2. Pilih periode pembayaran gaji
3. Sistem akan menampilkan daftar dokter dan staf beserta perhitungan:
    * Jam kerja
    * Jumlah pasien ditangani
    * Tindakan yang dilakukan
    * Bonus/insentif
4. Review perhitungan dan lakukan penyesuaian jika diperlukan
5. Klik **Proses Payroll** untuk finalisasi
6. Sistem akan menghasilkan slip gaji digital
7. Klik **Kirim Slip Gaji** untuk mengirimkan notifikasi ke staf

* Kepuasan pasien meningkat dengan sistem booking online

## 10. FAQ (PERTANYAAN UMUM)

### 10.1 Umum

**Q: Apakah data saya aman di CLIMA?**  
A: Ya, keamanan data adalah prioritas utama kami. CLIMA menggunakan enkripsi end-to-end untuk melindungi data pasien dan klinik. Semua server kami berlokasi di Indonesia dan mematuhi regulasi perlindungan data kesehatan.

**Q: Apakah saya bisa mencoba demo CLIMA sebelum berlangganan?**  
A: Tentu! Kami menyediakan versi demo gratis selama 14 hari dengan akses ke semua fitur. Hubungi admin kami untuk mengaktifkan akun demo.

**Q: Bisakah CLIMA digunakan oleh lebih dari satu pengguna secara bersamaan?**  
A: Ya, CLIMA mendukung akses multi-user. Anda bisa menambahkan akun untuk dokter, perawat, staf administrasi, dan kasir dengan level akses yang berbeda-beda.

### 10.2 Teknis

**Q: Apakah CLIMA memerlukan instalasi software khusus?**  
A: Tidak, CLIMA adalah sistem berbasis web yang dapat diakses melalui browser. Tidak diperlukan instalasi software tambahan.

**Q: Apa yang terjadi jika internet di klinik saya terputus?**  
A: CLIMA memiliki fitur offline mode yang memungkinkan Anda tetap mencatat data pasien. Data akan otomatis tersinkronisasi ketika koneksi internet kembali tersedia.

**Q: Bisakah saya mengimpor data dari sistem lama ke CLIMA?**  
A: Ya, kami menyediakan layanan migrasi data dari sistem lama (baik digital maupun manual) ke sistem CLIMA.

### 10.3 Biaya & Langganan

**Q: Apakah ada biaya setup atau instalasi?**  
A: Tidak ada biaya setup untuk paket standar. Untuk kebutuhan kustomisasi, mungkin ada biaya tambahan tergantung kompleksitas.

**Q: Apakah ada kontrak jangka panjang yang mengikat?**  
A: Tidak. Untuk paket bulanan, Anda dapat berhenti berlangganan kapan saja. Untuk paket tahunan, Anda terikat selama periode berlangganan (1 tahun).

**Q: Jika saya upgrade paket, apakah saya perlu membayar penuh?**  
A: Tidak, sistem kami menghitung biaya secara prorata. Anda hanya membayar selisih harga untuk periode yang tersisa.
